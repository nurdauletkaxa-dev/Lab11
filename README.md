Лабораторная Работа №11: Менеджер Задач

Это простое одностраничное приложение (SPA), реализованное с использованием библиотеки React. Приложение предназначено для создания, управления и отслеживания задач.

Основная Функциональность

Добавление задач: Возможность быстро ввести новую задачу через форму.

Удаление задач: Удаление любой задачи из списка.

Отметка о выполнении: Каждая задача имеет статус (выполнена/не выполнена), который можно переключить одним кликом.

Сохранение данных: Все созданные задачи автоматически сохраняются в LocalStorage браузера и восстанавливаются при повторном открытии приложения.

Используемые Технологии

React: Основная библиотека для построения компонентного пользовательского интерфейса.

JavaScript (ES6+): Для логики приложения и управления состоянием.

Tailwind CSS: Использован для быстрого и чистого стилизования компонентов.

Структура Проекта

Проект разделен на логические модули для удобства разработки и поддержки:

Файл

Описание

index.html

Базовый HTML-файл. Точка загрузки, подключает стили и главный скрипт.

main.js

Точка входа в React-приложение, монтирует главный компонент (App).

App.js

Главный компонент. Содержит общее состояние (tasks) и логику управления данными (добавление, удаление, переключение, LocalStorage).

TaskForm.js

Компонент для ввода текста и добавления новой задачи.

TaskList.js

Компонент-контейнер, который отображает список всех задач.

TaskItem.js

Компонент для отображения одной задачи с чекбоксом и кнопкой удаления.
